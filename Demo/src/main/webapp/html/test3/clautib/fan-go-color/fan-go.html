艺术的色彩语言

梵高用色彩诉说情感，每一种颜色都承载着他内心的激情与痛苦。从《星夜》的深邃蓝色到《向日葵》的热烈黄金， 从自画像中的强烈对比到卧室里的温暖和谐，这些配色方案不仅是视觉的盛宴，更是情感的表达。 让我们通过这些永恒的色彩组合，感受大师笔下的艺术魅力。
